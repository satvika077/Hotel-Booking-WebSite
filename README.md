# Hotel-Booking-WebSite
# 🏨 Hotel Booking Website

A full-stack web application inspired by Airbnb, where users can explore, list, and manage hotel bookings.

---

## 🚀 Features

* 🔐 User Authentication (Login / Register)
* 🏠 Add, Edit, Delete Hotel Listings
* 🔍 Browse and view hotel details
* 📍 Location-based listings
* 🖼️ Image uploads for properties
* 💬 Flash messages & validations

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript, EJS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Other Tools:** Cloudinary (for image upload), Map APIs

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/Hotel-Booking-WebSite.git
```

2. Navigate to the project folder:

```
cd MAJORPROJECT
```

3. Install dependencies:

```
npm install
```

4. Create a `.env` file and add:

```
MONGO_URI=your_mongodb_connection_string
SESSION_SECRET=your_secret_key
CLOUDINARY_KEY=your_cloudinary_key
```

5. Run the application:

```
npm start
```

6. Open in browser:

```
http://localhost:8080
```
